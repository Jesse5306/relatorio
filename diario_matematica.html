Par ordenado

Par ordenado é um par de objetos matemáticos cuja ordem de ocorrência desses objetos é signficante. Isso quer dizer, por exemplo, que um ponto qualquer no plano cartesiano (x, y) é diferente de (y, x). Nesse caso, se x = 2 e y = 3, o ponto (2, 3) é diferente de (3, 2). Além desse exemplo, o conjunto solução de um sistema linear com 2 icógnitas é também um par ordenado.

Produto cartesiano

Produto cartesiano é o conjunto de todos os pares ordenados dados dois conjuntos quaisquer (A e B), em que o primeiro termo pertence a A, e o segundo pertence a B. Em termos matemáticos, A x B = {(x, y)|x ∈ A e y ∈ B}. Exemplo: A = {3, 1} e B = {2, 7}, A x B = {(3, 2); (3, 7); (1, 2); (1, 7)}. Detalhe: a quantidade de elementos de um produto cartesiano é igual ao produto da quantidade de elementos de A com a quantidade de elementos de B devido ao princípio fundamental da contagem (QA x QB = QC, em que Q = quantidade).

Relação binária

Relação binária é uma relação entre dois conjuntos (A e B) e é um subconjunto do produto cartesiano entre A e B, em que há uma regra estabelecida que define essa relação. Em termos matématicos: R = {(x, y) ∈ A x B|[aqui onde se estabelece uma regra]}. Exemplo: Se A = {1, 2, 3, 4, 5} e B = {1, 2, 3, 4}, quais os elementos da relação R = {(x, y) ∈ A x B|x < y}? R = {(1, 2); (1, 3); (1, 4); (2, 3); (2, 4); (3, 4)}.

Relação inversa

Relação inversa é definida por R = {(y, x) ∈ B x A|[regra]}, para ser determinada, basta calcular a relação binária e inverter a ordem dos elementos que compõem os pares ordenados. Exemplo: Qual é a relação inversa de A e B no exemplo anterior? R = {(2, 1); (3, 1); (4, 1); (3, 2); (4, 2); (4, 3)}.

Introdução a funções

Definição de função: dados dois conjuntos não vazios A e B, uma relação binária que associa a cada elemento x ∈ A um único elemento y ∈ B recebe o nome de função de A em B.

* f é aplicação de A em B se, e somente se, (∀x ∈ A, ∃y ∈ B|(x, y) ∈ f) (Para todo x que pertence a A, existe pelo menos um y que pertence a B, tal que (x, y) pertence a f).

Condições

  1. É necessário que todo elemento x ∈ A participe de pelo menos um par (x, y) ∈ f, isto é, todo elemento de A "deve servir como ponto de partida da flecha".
  2. É necessário que cada elemento x ∈ A participe de apenas um único par (x, y) ∈ f, isto é, cada elemento de A "deve servir como ponto de uma única flecha".

Explicando com as próprias palavras

Imagine dois conjuntos, A e B, e dois pares ordenados, x e y, em que x pertence a A e y pertence a B, para ser considerado função é necessário que o mesmo x não pode dar dois valores de y. Por exemplo, A = {0, 1} e B = {1,4}, A x B = {(0, 1); (0, 4); (1, 1); (1, 4)}; o produto cartesiano não é função porque, se você olhar os pares ordenados dele, perceberá que o mesmo zero dá resultados diferentes (o zero de x dá um como resultado de y e também quatro, e o um de x dá um como resultado de y e também quatro), ou seja, o x não pode se repetir em qualquer par ordenado.

Entretanto, se A = {1, 2, 3}, B = {3, 5, 7} e C = {(x, y) ∈ A x B|y = 2x + 1}, C = {(1, 3); (2, 5); (3, 7)}. Repare que o conjunto C é uma relação binária que respeita a regra no parágrafo acima, pois o x não se repete.

Outra coisa que vale salientar é que todos os elementos de A devem ser usados, mas nem todos os elementos de B precisam ser usados (pelo menos um deve). Aqui está um video que deve explicar melhor: vídeo. Nele, o professor Rafael exemplifica o assunto, muito melhor do que ficar só na definição.

Funções númericas

Funções numéricas são relações matemáticas que associam elementos de dois conjuntos. Em termos simples, uma função pega um valor de um conjunto (chamado de domínio) e o relaciona a um único valor de outro conjunto (chamado de contradomínio). A notação comum para uma função é ( f: A \seta para direita B ), onde cada elemento ( x ) do conjunto ( A ) é mapeado para um elemento ( y ) do conjunto ( B ) através da função( f ), que é basicamente o que eu tinha explicado, com uma única diferença: é preciso uma lei de formação, e y = f(x).

Função do primeiro grau

A função do primeiro grau, também conhecida como função afim, é uma função númerica que pode ser expressa na forma y = ax + b, onde a e b são números reais e a ≠ 0. Esta função é representada graficamente por uma reta.

    Características Principais:
  • Coeficiente Angular (a): Define a inclinação da reta. Se a > 0, a função é crescente; se a < 0, a função é decrescente.
  • Coeficiente Linear (b): Indica o ponto onde a reta intercepta o eixo \(y\).
    Exemplos:
  1. y = 2x + 3 (Função crescente)
  2. y = -x + 5 (Função decrescente)

Gráfico: o gráfico de uma função do primeiro grau é sempre uma linha reta. A inclinação e a posição da reta dependem dos valores de a e b. Ver imagem mais detalhada.

Caso 1: Função constante

Se a de y = ax + b = 0 é igual a 0, y = b. Nesse caso, a função será constante, isto é, o valor de y nunca irá mudar e sempre será igual ao coeficiente linear. Faz sentido, já que a é o coeficiente angular que determina a inclinação da reta em relação ao eixo x; se ele for igual a 0, a reta, no plano cartesiano, é horizontal. Ver imagem mais detalhada.

Caso 2: Função identidade

Numa função identidade, o coeficiente angular é sempre igual a 1, e o coeficiente angular é igual a 0. Isso significa que para cada unidade que você se move ao longo do eixo x , você se move exatamente uma unidade ao longo do eixo y. Matematicamente, a função identidade é representada por y = x , onde o coeficiente angular é o número que multiplica x, que neste caso é 1. Ver imagem mais detalhada.

Caso 3: Função linear

Quando o coeficiente angular é diferente de 0, e o coeficiente linear é igual a 0, chamamos essa função de função linear. Ver imagem mais detalhada.Exemplos de funções lineares: y = 2x (função dos números pares); y = -x; y = 10x;

Estudo do sinal

    O estudo do sinal da função do primeiro grau, também conhecida como função afim, é uma análise importante para entender onde a função é positiva, negativa ou nula. Vamos considerar uma função do tipo y = ax + b , onde a e b são números reais e a diferente de 0.

  1. Primeiro, calcule a raiz da função quando y = 0. ax + b = 0 -> ax = 0 - b = -b -> x = -b/a;
  2. Segundo, se a função é crescente (a > 0), y será positivo (y > 0) se x > -b/a, y será negativo (y < 0) se x < -b/a e y = 0 se x = -b/a;
  3. Terceiro, se a função é decrescente (a < 0), y será positivo se x < -b/a, y será negativo se x > -b/a e y = 0 se x = -b/a.

Raiz da função

A função do 2º grau, também chamada de função quadrática, possui a seguinte lei de formação: ax² + bx + c = 0, em que a é o coeficiente angular, ele determina a concavidade da parábola (se é maior que zero, a concavidade é voltada para cima; se é menor que zero, a concavidade é voltada para baixo), b é o coeficiente linear, ele influencia a inclinação da parábola que a função quadrática desenha no plano, e o c é o termo indepedente, é onde a parábola intercepta o eixo y no plano cartesiano. A fórmula que fornece o valor de x quando y é igual a zero é:

-b ± √Δ
2a

Exemplo: Qual é a raiz da função x² + 2x = 0 ? Resposta: Observando a imagem correspondente, percebe-se que a função intercepta o eixo x nos pontos (x=0, y=0) e (x=-2, x=0) e, analisando esses pontos, conclui-se que as raízes da função é 0 e -2. No entanto, nem sempre se tem acesso ao gráfico da função, então é aí que entra fórumla:

x' = -b + √(b² - 4ac) / 2a -> x' = -2 + √(2² - 4 × 1 × 0) / 2 × 1 -> x' = -2 + √(4 - 0) / 2 -> x' = -2 + 2 / 2 -> x' = 0 / 2 -> x' = 0

x" = -b - √(b² - 4ac) / 2a -> x" = -2 - √(2² - 4 × 1 × 0) / 2 × 1 -> x" = -2 - √(4 - 0) / 2 -> x" = -2 - 2 / 2 -> x" = -4 / 2 -> x" = -2

Coordenadas do vértice

Vértice é o ponto mínimo ou máximo da função, representa o valor mais baixo de y se a > 0 e representa o valor mais alto de y se a < 0. As fórmulas que fornecem as coordenadas são as seguinte:

-b

-Δ
4a

Exemplo: Qual são as coordenadas do vértice na função y = x² + 2x ? Resposta: Se você observar a imagem correspondente, verá que as coordenadas do vértice são xv = -1 e yv = -1. Porém, se você não tiver acesso ao gráfico da função, você tem que utilizar as fórmulas citadas:

xv = -b / 2a -> xv = -2 / 2 × 1 -> xv = -2 / 2 -> xv = -1

yv = -Δ / 4a -> yv = -(b² - 4ac) / 4a -> yv = -(2² - 4 × 1 × 0) / 4 × 1 -> yv = -4 / 4 -> yv = -1
